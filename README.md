# Data Analytics Portfolio

This repository showcases **end-to-end analytics projects** using **Python, SQL Server, and Power BI**.

Each project starts from raw data and ends with business insights, actionable recommendations, and interactive dashboards.

---

## Featured Projects

### 1. Customer Shopping Behaviour - Customer & Sales Intelligence

**Objective:** Understand revenue drivers, evaluate discount & subscription effectiveness, and optimize customer targeting.

**Tech Stack:**
- Python (pandas) - Data cleaning & feature engineering
- SQL Server - Data modeling & analytical queries
- Power BI - Interactive dashboards & storytelling

**Dataset:** ~3,900 customer transactions

**Key Deliverables:**
- 10 business-focused SQL queries
- 4-page Power BI dashboard
- Business recommendations

**Folder:** [`customer-shopping-behaviour/`](./customer-shopping-behaviour)
